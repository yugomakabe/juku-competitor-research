# スタイルガイド：教育産業競合調査レポート

## 文体

- 敬体（です・ます調）を使用する
- 断定的な表現より「〜とみられる」「〜とされる」「〜という評判がある」を使う（事実確認できない情報・口コミ由来の情報の場合）

## 数値の表記

- 料金：「月額○円（税込/税抜を明記）」のように税区分を必ず併記する。学年・コースにより異なる場合は代表的な例を挙げたうえでその旨を明記する
- 校舎数・生徒数：取得年度を併記する（例：「2025年時点で○校舎」）
- 合格実績：対象とする入試区分（中学受験／高校受験／大学受験）を明記したうえで、「○年度 ○○高校合格△名」のように年度・進学先・人数をセットで記載する
- 年度：西暦（例：2025年度）で統一し、和暦のみの表記は避ける

## 情報の信頼性

- 対象企業の公式サイト・IR/決算情報（該当する場合）を優先する
- 口コミサイト・SNS等の二次情報は、地域における認知・評判セクションでのみ、出典を明記したうえで参考情報として利用する
- 一次情報で確認できなかった項目は「（情報なし）」と明記する（空白にしない）
- 出典によって数値に幅がある場合は、単一の断定値にせず幅・出典の違いを明記する

## 「再就職検討への示唆」セクションの書き方

- 特定の企業への応募を推奨・非推奨するような断定的な結論は書かず、あくまで比較検討のための材料（強み・弱み・特徴の整理）として記述する
- 各社のデータ（運営概要・コース・料金・合格実績・DX施策・評判）から読み取れる特徴に基づいて記述し、憶測を書かない
